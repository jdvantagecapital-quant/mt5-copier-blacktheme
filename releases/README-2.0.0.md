# JD MT5 Trade Copier 2.0.0 (Windows)

- Build: dist/JD_MT5_TradeCopier.exe
- UI: Settings page redesigned with scoped styles
- Spec: SPEC_DIR fixed to current project path

## Run
Double-click the EXE or run:
- Start-Process .\dist\JD_MT5_TradeCopier.exe

Server listens on 127.0.0.1:5000.

## Notes
- Requires Windows with MT5 and Python deps bundled via PyInstaller.
- For clean repo, dist/ and build/ are ignored.
